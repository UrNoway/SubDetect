# SubDetect

SubDetect use FFmpeg, Leptonica and Tesseract to try getting subtitles in video proccessing.

Goals:
- [x] Use FFmpeg and Leptonica for processing video
- [ ] Use Tesseract to get text (work in progress...)
- [ ] Conventional grayscale filter
- [ ] Parallel filter
- [ ] Deliver subtitles formats (SRT by default)
